## ◈ What Lives Here

This is not just a folder of code. It's a **complete chronicle** of Semester 1 — every concept wrestled with, every bug that taught something, every line written in the pursuit of understanding how computers *actually* think.

Written entirely in **C++**, this repository archives the full arc: from a blinking cursor printing `Hello, World!` to real-world-esque problem solving with pointers, memory, and data structures. It's the foundation everything else will be built on.

## ◈ Chapter Breakdown

<details>
<summary><b>🟢 01 — Fundamentals &nbsp;|&nbsp; The Ground Floor</b></summary>
<br/>

The very start. Every concept here was a first encounter — syntax, structure, and the discipline of typed, compiled code.

| Topic | What Was Covered |
|---|---|
| **Hello World** | Entry point, `cout`, compilation pipeline |
| **Variables & Types** | `int`, `float`, `double`, `char`, `bool`, type casting |
| **Conditionals** | `if / else if / else`, nested conditions, `switch` |
| **Loops** | `for`, `while`, `do-while`, nested iterations |
| **Arrays** | 1D & 2D arrays, traversal, manipulation |


> This chapter stretches from the very first line of C++ all the way through arrays — a complete grip on the language's bedrock.

</details>

<details>
<summary><b>🟡 02 — Advanced Logic &nbsp;|&nbsp; Going Deeper</b></summary>
<br/>

Where things got serious. This chapter is about understanding what the machine is *actually doing* — memory addresses, dynamic allocation, and how data is truly structured.

| Topic | What Was Covered |
|---|---|
| **Pointers & References** | Address-of, dereferencing, pointer arithmetic |
| **Memory Allocation** | `new` / `delete`, heap vs stack, memory leaks |
| **Strings** | `std::string`, C-strings, manipulation functions |
| **Recursion** | Base cases, call stacks, recursive problem solving |
| **Structures** | `struct` design, nested structs, real-world modeling |


> Real-world-inspired problems were applied throughout — this isn't dry theory, it's logic put to work.

</details>

<details>
<summary><b>🚀 03 — Minor Projects &nbsp;|&nbsp; Build Something Real</b></summary>
<br/>

Standalone applications and tools built to consolidate everything from the first two chapters. Each project is a contained, working program — proof that the concepts weren't just memorized, they were *used*.

</details>

<details>
<summary><b>🏆 04 — Problem Solving &nbsp;|&nbsp; Sharpen the Mind</b></summary>
<br/>

Logic-building exercises and competitive-programming-style problems. This is where patterns are recognized, efficiency is considered, and thinking in algorithms becomes second nature.

</details>

## ◈ How to Explore

```bash
# Clone the vault
git clone https://github.com/MuhammadAhmadHamim/cpp-core-foundations.git
cd cpp-core-foundations

# Navigate to any chapter
cd 01_Fundamentals

# Compile any file with g++
g++ -std=c++17 -Wall -o output filename.cpp

# Run it
./output
```


> **Compiler:** GCC / G++ recommended &nbsp;|&nbsp; **Standard:** C++17 preferred &nbsp;|&nbsp; **IDE:** VS Code, CLion, or any terminal
